Understanding Builders

This documentation guides you through the various builder options available in your angular.json file. Builders in Angular are responsible for crucial tasks in your project, such as serving, building, and testing. Each builder has its own set of configurable options that you can customize to fit your project's needs.

This documentation supports Angular versions 12 through 18 and covers various builder options available in Angular. Common Builders include, but are not limited to:

  • settings Application Configures the overall application build process.
  • web Browser Handles browser-specific build configurations.
  • dns Server Manages server-side rendering configurations.
  • computer Dev-Server Controls the development server settings.

Configuring Properties

To configure a builder, locate its section in your angular.json file and adjust the properties as needed. For example:


  {
    "projects": {
      "your-app": {
        "architect": {
          "build": {
            "builder": "@angular-devkit/build-angular:browser",
            "options": {
              // Builder properties go here
            }
          }
        }
      }
    }
  }
