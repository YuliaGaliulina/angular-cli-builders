Welcome to the unofficial Angular CLI builders documentation!

Understanding Builders

This documentation is generated dynamically from the builder schemas available in the Angular CLI, providing a comprehensive outline of the various builder options in your angular.json file. Angular builders handle crucial project tasks like serving, building, and testing. Each builder offers configurable options you can customize to fit your project's needs.

This documentation supports Angular versions 12 through 18.

Configuring Properties

To configure a builder, locate its section in your angular.json file and adjust the properties as needed. For example:


  {
    "projects": {
      "your-app": {
        "architect": {
          "build": {
            "builder": "@angular-devkit/build-angular:browser",
            "options": {
              // Builder properties go here
            }
          }
        }
      }
    }
  }
